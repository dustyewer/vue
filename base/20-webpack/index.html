坑爹的版本不兼容，不能使用默认的版本，以下是验证可行的版本组合

webpack@3.6.0

css-loader@0.28.7

style-loader@0.23.1

less@3.9.0

less-loader@4.1.0

url-loader@1.1.2

file-loader@2.0.0

es6 转 es5 ,还得装babel;命令:npm install --save-dev babel-loader@7 babel-core babel-pres et-es2015

安装上面的loader之后，还得配置webpack.config.js

正式引入vue 开发,运行环境都要使用vue,不要--save-dev,要--save, 命令:npm install --save vue@2.5.21

runtime-only版本 不可以用任何template runtime-compiler版本 可以有template
